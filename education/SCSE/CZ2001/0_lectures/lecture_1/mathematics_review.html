Mathematical Background

Sets

A set is a collection of well defined and distinct objects.

A relation (x,y) in a set A is a collection of ordered pairs of elements in A.

Functions

A function is a relation between a set of inputs and a set of potential outputs with the property that each input is related to exactly one output.

Formal description of a function typically involves the function's name, its domain, its codomain, and a rule of correspondence.

  • e.g. f : ℜ → ℜ, f(x) = x2

The floor and ceiling functions map a real number to the largest previous or the smallest following integer, respectively.

  • ⌊x⌋ (floor of x): the largest integer not greater than x (i.e. round-down / truncate)
  • ⌈x⌉ (ceiling of x): the smallest integer not less than x (i.e. round-up)
  • ⌊x⌋ ≤ x ≤ ⌈x⌉

Power Function (Exponentiation)

Exponentiation is a mathematical operation, written as bn, involving two numbers, the base b and the exponent (a.k.a. index or power) n.

When n is a positive integer, exponentiation corresponds to repeated multiplication.
