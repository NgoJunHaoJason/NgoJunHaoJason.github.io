Mathematical Background

Sets

A set is a collection of well defined and distinct objects.

A relation (x,y) in a set A is a collection of ordered pairs of elements in A.

Functions

A function is a relation between a set of inputs and a set of potential outputs with the property that each input is related to exactly one output.

Formal description of a function typically involves the function's name, its domain, its codomain, and a rule of correspondence.

  • e.g. ƒ : ℜ → ℜ, ƒ(x) = x2

The floor and ceiling functions map a real number to the largest previous or the smallest following integer, respectively.

  • ⌊x⌋ (floor of x): the largest integer not greater than x (i.e. round-down / truncate)
  • ⌈x⌉ (ceiling of x): the smallest integer not less than x (i.e. round-up)
  • ⌊x⌋ ≤ x ≤ ⌈x⌉

Power Function (Exponentiation)

Exponentiation is a mathematical operation, written as bn, involving two numbers, the base b and the exponent (a.k.a. index or power) n.

When n is a positive integer, exponentiation corresponds to repeated multiplication.

  • b0 = 1
  • b1 = b
  • b2 = b(b)
  • ...
  • bn = b(bn-1)

Logarithm Function

The logarithm of a number to the base b is the exponent by which the base b has to be raised to produce that number.

  • y = logb x ⇔ x = by

formulas:

  • logb 1 = 0 (since b0 = 1)
  • logb 0 is undefined (since b? = 0)
  • logb xn = n logb x
  • logb xy = logb x + logb y
  • logb x = logc x / logc b

Summations and Series

A series is, informally speaking, the sum of the terms of a sequence.

  • finite series: S = 1 + 2 + 3 + ... + n = n(n + 1) / 2
  • infinite series: S = 1 + ½ + ¼ + ⅛ + ... = ∑n→∞1/2n

Arithmetic Series

Each successive term is produced by adding a constant number to the previous term.

 n-1∑i=0 (a + id)
= a + (a + d) + (a + 2d) + ... + (a + id) + ... + [a + (n-1)d]
= n/2[2a + (n-1)d]
= n/2(a + L)
where:

  • n = number of terms
  • a = first term
  • d = common difference
  • L = last term

Geometric Series

Each successive term is produced by multiplying the previous term by a constant number.

 n-1∑i=0 ari
= a + ar + ar2 + ... + ari + ... + arn-1
= a(1 - rn / 1 - r)
= a(rn - 1 / r - 1)
where:

  • n = number of terms
  • a = first term
  • r = common ratio

Differentiation

Differentiating a function is to find the derivative or rate of change of the function.

formulas:

  • d/dx cxn = cnxn-1
  • d/dx bf(x) = bf(x) ln b f'(x)
  • d/dx logb f(x) = f'(x) / f(x) ln b
