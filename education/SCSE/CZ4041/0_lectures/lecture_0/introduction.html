Introduction

What is machine learning?

Machine learning focuses on the development of computer programs that can teach themselves to grow from data and change when exposed to new data.

It is motivated by how human beings learn from examples, experience or exercise.

Motivating example

Given an image of a person's face, classify the person's sex.

In the past, to develop AI system to solve such a task, developers or domain experts need to provide rules (if-then) and implement them in the system.

example rules:

  • If the person's face has long hair and does not have moustache, then the person is female.
  • If the person has short hair and a moustache, then the person is male.

limitations:

  • time consuming - have to define rules for each domain
  • the defined rules may be incomplete - cannot handle case where face has long hair and moustache.
  • unable to handle uncertainty - in real world, outcome is usually based on probability, so a lot of cases cannot be handled

How about letting the machine learn the rules by itself?

  • The computer is presented with example inputs and their desired outputs, given by human.
  • The goal is to learn a general rule (or model) that maps inputs to outputs.

definition of machine learning

Machine learning is a type of artificial intelligence that provides computers with the ability to learn from examples or experience without being explicitly programmed.

how to represent an example

  • feature engineering (not the focus of machine learning)
  • representation learning (one of the crucial research topics in machine) - deep learning is currently the most effective form of representation learning

relationship between machine learning and related keywords

Machine learning is a field of artificial intelligence - there are many other fields of artificial intelligence.

Deep learning is a methodology of machine learning - there are many other methodologies in machine learning.

Machine learning has become the primary mechanism for data analytics (which is key in data science).

Nowadays, machine learning is more interdisciplinary:

  • distributed / parallel computing + machine learning → distributed / parallel machine learning
  • machine learning + hardware → neural processors

machine learning paradigms

  • supervised learning
  • unsupervised learning
  • reinforcement learning
  • advanced paradigms:
    • semi-supervised learning
    • active learning
    • transfer learning

supervised learning

The examples presented to computers are pairs of inputs and corresponding outputs (labelled training data). The goal is to learn a mapping or model from inputs to labels.

  • outputs are discrete values → classification
  • outputs are continuous values → regression

unsupervised learning

The examples presented to computers are a set of inputs without any outputs. The goal is to learn an intrinsic structure of the examples (e.g. clusters of the examples, density of the examples).

reinforcement learning

An agent learn by interacting with an environment to achieve a goal. The objective is to learn an optimal policy mapping states to actions.

semi-supervised learning

The examples presented to computers include both labelled data and unlabelled data. The goal is to us unlabelled data to aid supervised learning.

active learning

The examples presented to the computer are unlabelled. The active learner (computer) may pose queries in the form of unlabelled examples to be labelled by an oracle (human annotator) with cost.
