Naïve Bayes

independence

Let A and B be 2 sets of random variables.

The variables in A are said to be independent of B if P(A, B) = P(A | B) × P(B) = P(A) × P(B)

conditional independence

Let A, B and C be 3 sets of random variables.

The variables in A are said to be conditionally independent of B, given C, if P(A | B, C) = P(A | C).

The conditional independence between A and B given C can be written as:
   P(A, B | C)
= P(A, B, C)/P(C)
= P(A, B, C)/P(B, C) × P(B, C)/P(C)
= P(A | B, C) × P(B | C)
= P(A | C) × P(B | C)

naïve Bayes classifier

Assuming that the features are conditionally independent given the class label, the likelihood can be estimated as P(X | Y = yk) = d∏i=1 P(Xi | Y = yk), where X = [X1, X2, ..., Xd].
